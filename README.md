## 📌 About the Project

**SURFIN' BIRD QUEST PREMIUM** is a front-end arcade game built around speed, rhythm, and order handling. The visible game UI shows a kitchen area where players **tap items to build a tray**, then use actions like **Serve Order** or **Clear** while managing score, combo, wave progression, queue pressure, and lives. :contentReference[oaicite:1]{index=1}

This project combines:

- 🐦 arcade-style scoring

- ❤️ life management
- 🏆 leaderboard and achievements

## 🕹️ Gameplay Overview

The visible interface suggests a fast reaction loop centered on preparing and serving orders.

### 1. Build the Tray
The game instructs the player with the message:

**“Kitchen — tap items to build your tray”**  
and shows a dedicated **Tray** area for assembling the current order. :contentReference[oaicite:3]{index=3}

### 2. Serve or Clear
Once the tray is prepared, the player can choose:

- **✓ Serve Order**
- **✕ Clear** :contentReference[oaicite:4]{index=4}

This implies a quick decision cycle where accuracy and timing likely matter for score and combo retention.

### 3. Survive the Waves
The UI includes:

- **Score**
- **Combo**
- **Wave**
- **Queue**
- **Lives**
- **LV 1** :contentReference[oaicite:5]{index=5}

That structure strongly suggests a progressive arcade format where pressure increases over time as the player advances through waves.

### 4. Use Power-Ups
A dedicated section states:

**“Power-Ups — spend score to activate”** :contentReference[oaicite:6]{index=6}

This adds a risk/reward layer where score becomes both a success metric and a gameplay resource.
